# CrackShot-clj
Clojureで書かれたCrackShot(もどき)です。

## 差異

- アイテムIDは使えない。Materialのみ受け付けるように
- SOUNDの書式は(ほぼ)使えない。Soundのみ受け付けるように
- POTIONTYPEは使えない。PotionEffectTypeのみ受け付けるように

- 

## 互換性無し

- SignShop
- Crafting
- Region_Check
- Spawn_Entity_On_Hit
- Abilities
  - Jet_pack
- Extras
  - Make_Victim_Run_Commmand
  - Run_Console_Command
  - Run_Command
